AI basic: Knowledge

三月 16, 2024, 12:54 下午

Knowledge

与人类类似，AI也能通过现有知识“推理”出新的知识。

如何将“知识”数据化存储到AI中?

  • Sentence

    A sentence is an assertion about the world in a knowledge representation language.

    Example:

    SYSU is a college —> College(sysu)

    Timmy is a student in SYSU —> Student(timmy, sysu)

简单来说，AI基于“一阶逻辑”(First Order Logic)符号体系来对知识进行存储。

一阶逻辑是一种形式系统，用符号来表示命题、谓词、量词和逻辑连接诸如与、或、非等。它包括一个语言部分，用于描述命题和量词的含义，以及一个推理规则部分，用于推断命题之间的关系。一阶逻辑通常被用来进行严格的数学和哲学推理。

Knowledge Base

Knowledge Base (知识库) 可以看成一个 Sentence 的集合。

其中每个元素都是经过符号化处理后的 Sentence，代表我们已有的知识。在知识库的基础上，AI可以对新的知识进行推理。

我们可以将知识推理看作一个搜索问题: 我们尝试从知识库KB中推理Sentenceα

  • Initial State: KB
  • Actions: 各种推理规则
  • Transition model: 经过推理加入新知识后的新的知识库
  • Goal test: 判断我们询问的目标α是否存在于KB中。
  • Path cost: 得到α的过程中推理的次数

下面介绍两种最基本的推理算法: Model Checking 与 Resolution

Model Checking

一言以蔽之，本算法通过“真值表”来直接实现推理。

即遍历整个”真值表”，若KB->α是永真式，则表明我们可以从KB中推导出α；反之说明不能。

Resolution

消解
本算法基与以下很好证明的却又没那么显而易见的推理规则:

实质上是反证法: (KB⊨α) 与 (KB∧¬α) 是矛盾的。只要证明后者不成立(False),就能证明前者成立(True)

$(P) ∧ (¬P) ⊨ () $

$(P ∨ Q) ∧ (¬P) ⊨ Q$

$(P ∨ Q) ∧ (¬P ∨ R) ⊨ (Q ∨ R)$

基本步骤如下:

  1. 将 (KB∧¬α) 转换为合取范式
  2. 遍历范式中的每个子句，不断尝试用 消解规则 产生新的子句
    1. 若产生的新子句是空子句()，说明(KB∧¬α)为假，进而表示 (KB⊨α)为真，即我们能从KB中推理出α。
    2. 若无法继续产生新的子句，则说明我们不能从KB中推理出α。

入东校有感

八月 27, 2023, 10:02 上午

这也许是我第一次主动写作，第一次产生“想写些什么记录一下我此刻的心情”的想法，不同与以往“应试”的写作。这实在是难得。
为什么突然会有这种想法？也许是我在人生中第一次产生了“孤独”的感觉。
我并不承认在此之前我感到孤独过。“孤独”可以分成“孤”和“独”两个字看，“独”是一种状态，而“孤”是由“独”引发的感受。往常我在“独”这种状态下并没有产生过“孤”的感受，反而很享受一个人的感觉。也许是之前过着集体生活，无比渴望个人空间，但凡身边有人由于情绪波动说话声音稍微大了点，我心里都会不愉快（尽管面上不显）。
但大二搬到东校区后，换了专业，换了宿舍，换了一批新同学新老师，我的人际关系近乎清空，我常常是一个人，干什么都是一个人，独自吃饭，独自学习，独自娱乐，我竟感到有些空虚了（这是我过往“梦寐以求”的状态）。确实是空虚，感到整个人都是空空的，想找点事做，却又什么都不想做，心里空落落的，又找不出原因，“也许这就是‘孤’的感觉吧”，我想。就这样，我自认为第一次感到了孤独。这感受无以排挤，我又不想整日荒废在电子游戏中，故想写些什么，来尝试排挤我的孤独。
相对与东校区，我大一所在的南校区可谓人满为患。南校学生多，所有的大一新生都在南校区，还有很多的高年级学生与研究生；闲杂人等也多，到处都是教职工的家属与来参观游览的“游客”，很多时候某些场合学生的人数甚至不够他们多。人多了自然就吵闹，吵闹了自然就引发我的不适。所以大一在南校区待着的一年，我都很嫌弃它。相对而言，东校似乎更适合我，我早早就期待着去往东校：东校面积很大，教学区与生活区相互分开，学生密度相对较小，大学城也清净，学校里没什么“闲杂人等”，最适合我这种社恐。
但实际上到了东校才发现，我还是有些渴望社交的。我的几个新舍友都挺好，我们性格相近，相处融洽，美中不足的是，太安静了。我自己都觉得很矛盾，我大一的舍友相对而言话多很多，有时候我也觉得烦，想着“要是给我凑几个没话的舍友就好了”；但现在新舍友真的没话了（我算是全宿舍话最多的人），我反倒又怀念起大一那些话多的舍友了。也许这也是我感到“孤独”的原因之一，我接触到的“话”太少了。以前有时候我一天也不跟别人说几句话，但其他人在我周边说话，我不可避免地听他们说，接受他们发出的信息（尽管那些话不是对我说的）。而现在，我身边都没几个人说话，我也没能听见别人说多少话，我也没说几句话，一天下来，我都没接触到多少“话”，因此产生了空虚感。也许是因为我已经习惯了“话”多的环境，突然到了“话”少的东校，水土不服了。如果这个假设成立的话，在开学之后我的“孤独”感应该会缓解不少–上课总是要听老师讲话的。
希望如此。
当然还有一种可能是我放假太闲了，一忙起来什么孤独都没了。

附：《夜坐记》一篇，作者是沈周。
寒夜寝甚甘，夜分而寤，神度爽然，弗能复寐。乃披衣坐起，一灯荧然相对，案上书数帙，漫取一编读之；稍倦，置书束手危坐，久雨新霁，月色淡淡映窗户，四听阒然。盖觉清耿之久，渐有所闻。
闻风声撼竹木号号鸣，使人起特立不回之志；闻犬声狺狺而苦，使人起闲邪御寇之志；闻小大鼓声，小者薄而大者渊渊不绝，起幽忧不平之思；官鼓甚近，由三挝以至四至五，渐急以趋晓，俄东北声钟，钟得雨霁，音极清越，闻之有待旦兴作之思，不能已焉。
余兴喜夜坐，每摊书灯下，反复之，迨二更以为当。然人喧未息而又心在文字间，未尝得外静而内定。而于今夕者，凡诸声色，盖以定静得之，故足以澄人心神情而发其志意如此。且他时非无是声色也，非不接于人耳目中也，然形为物役而心趣随之，聪隐于铿訇，明隐于文华，是故物之益于人者寡而损人者多。
有若今之声色不异于彼，而一触耳目，犂然与我妙合，则其为铿訇文华者，未始不为吾进修之资，而物足以役人也已。声绝色泯，而吾之志冲然特存，则所谓志者果内乎外乎，其有于物乎，得因物以发乎？是必有以辨矣。于乎吾于是而辨焉。夜坐之力宏矣哉！嗣当齐心孤坐，于更长明烛之下，因以求事物之理，心体之妙，以为修己应物之地，将必有所得也。
作《夜坐记》。
弘治壬子秋七月既望，长洲沈周。

这也许就是我理想中“独而不孤”的状态。

记录一下搭建过程的一些BUG

八月 26, 2023, 3:30 下午

在创建与配置本博客时遇到的一些问题与解决方案

2023-08-26

遇到问题：更换volantic主题后上传不完全。

Solution: 每次提交时用 hexo clean && hexo deploy 先清除缓存。
但Edge浏览器刷新了很多次上传仍不完全，看了issue后换了Chorme试试，结果正常了。
访问建议开无痕模式。

2023-09-07

遇到问题：服务器上传文章时图片加载失败。我在本地的md文件渲染后能看到，但上传后LOCALHOST里都加载不出来。

Solution: 将Github作为图床配合PicGo管理，将需要的图片上传到GitHub上，再在文章中进行引用。最终可以显示。

2023-10-10

遇到问题：无法更换封面

Solution: 配置文件中 background 多缩进了一格，取消缩进后恢复正常
