Uncertainties

note for Havard CS50.AI lecture02

Uncertainties,即不确定性，第一眼看到这个标题我并没能理解他的意思。学完后再回看，才了解，这一节包含了概率论的一些基本的内容。

Random Variables

这里讨论的是“离散型随机变量”

For example, to represent the status of a flight, we can define a variable Flight that takes on the values {on time, delayed, canceled}.

Often, we are interested in the probability with which each value occurs. We represent this using a probability distribution.

  • P(Flight = on time) = 0.6
  • P(Flight = delayed) = 0.3
  • P(Flight = canceled) = 0.1

Then this probability distribution can be represented more succinctly as a vector:

  • P(Flight) = <0.6, 0.3, 0.1>

Bayes’ Rule

Simply review it,

  • P(b|a)=P(b)P(a|b)/P(a)

    In words, Bayes’ rule says that the probability of b given a is equal to the probability of a given b, times the probability of b, divided by the probability of a.

Joint Probability

Simply review it,

  • P(b|a) = P(a, b) / P(a)

    In probability, commas and ∧ are used interchangeably.

If P(a) is viewed as a constant by which P(a, b) is multiplied, we can re write P(b|a) as αP(a, b)

Marginalization

Simply review it,

  • P(a) = P(a, b) + P(a, ¬b)

Or in a more general way:

  • $P(X=x_{i})=\sum_{j}P(X=x_{i}, Y=y_{j})$

Conditioning

Simply review it,

  • P(a) = P(a | b)P(b) + P(a | ¬b)P(¬b)

Or in a more general way:

  • $P(X=x_{i})=\sum_{j}P(X=x_{i}|Y=y_{j})P(Y=y_{j})$

Bayesian Networks

A Bayesian network is a data structure that represents the dependencies among random variables.

  • They are directed graphs. (DAG)
  • Each node on the graph represent a random variable.
  • An arrow from X to Y represents that X is a parent of Y. That is, the probability distribution of Y depends on the value of X.
  • Each node X has probability distribution P(X | Parents(X)).

一般而言，贝叶斯网络的有向无环图中的节点表示随机变量，它们可以是可观察到的变量，抑或是隐变量、未知参数等。连接两个节点的箭头代表此两个随机变量是具有因果关系或是非条件独立的；而两个节点间若没有箭头相互连接一起的情况就称其随机变量彼此间为条件独立。若两个节点间以一个单箭头连接在一起，表示其中一个节点是“因（parents）”，另一个是“果（descendants or children）”，两节点就会产生一个条件概率值。

Inference

We can infer new information based on probabilities. While this does not allow us to know new information for certain, it allows us to figure out the probability distributions for some values.

Inference has multiple properties:

  • Query X
  • Evidence variables E
  • Hidden variables Y
  • Goal: P(X|e)

    Namely, to compute the probability distribution of X based on evidence e.

There is an algorithm to compute this by enumberation:

  • $P(X|e) = αP(X,e) = α \sum_{y} P(X,e,y)$

    X : query variable ; e : observed evidence,
    y : hidden variables

Markov Model

First we look at an assumption. Then how the model works.

Markov Assumption

The current state depends on only a finite fixed number of previous states.

Markov Chain

A Markov chain is a sequence of random variables where the distribution of each variable follows the Markov assumption. That is, each event in the chain occurs based on the probability of the event before it.
